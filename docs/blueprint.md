# **App Name**: Batfolio Firebase

## Core Features:

- Firebase Authentication: Implement Firebase Authentication (Email Link or Phone OTP) to replace local OTP simulation, securing edit mode access.
- Firestore Integration: Replace localStorage with Firestore for real-time, scalable portfolio data storage.
- Firestore Security Rules: Define robust Firestore rules to protect user data and prevent unauthorized access.
- Scalable Data Schema: Design a Firestore data schema that supports future scalability and new features.
- UI Preservation: Maintain the existing dark mode, theme toggles, and animations with minimal code changes during Firebase integration.
- Viewer Mode: A way to show your site to the world in viewer mode. Activate this tool, if you are logged in.
- Interactive Elements: Interactive elements, including custom cursor and theme toggle.
- Security Measures: Prevent right-click context menu and DevTools shortcuts.

## Style Guidelines:

- Primary color: Dark Blue (#1E3A8A) to convey professionalism and trust.
- Background color: Very Dark Gray (#0D0D0D) to maintain a sleek, modern feel with a dark theme.
- Accent color: Purple (#9370DB) to highlight interactive elements and add a touch of sophistication.
- Headline font: 'Space Grotesk' sans-serif for a modern, techy look. Body font: 'Inter' sans-serif to ensure readability.
- Use simple, outlined icons in a single style across the app. These should be white or light gray for contrast.
- Employ a minimalist layout with clear sections and plenty of whitespace for easy navigation.
- Subtle animations and transitions to enhance user experience. Include a custom cursor for viewer mode.
